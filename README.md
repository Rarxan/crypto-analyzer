# **Crypto Analyzer**

A simple and powerful tool for encryption and decryption based on the Caesar Cipher. It includes functionality for brute-force decryption and file handling.

---

## ✅ Features

- 🔐 **Encryption** — Encrypt any text with a specified key using Caesar Cipher  
- 🔓 **Decryption** — Decrypt any text with a specified key  
- 💣 **Brute-force Decryption** — Automatically tries all possible keys to find the correct one  
- 📂 **File Input/Output** — Encrypt and decrypt files directly  
- 🖥️ **Command Line Interface** — Simple console usage, no extra setup needed  

---

## 🚀 Installation
```bash
git clone https://github.com/Rarxan/crypto-analyzer.git
cd crypto-analyzer
mvn clean install
```
---

▶️ Usage
java -jar target/crypto-analyzer.jar

Once the program is running, select the desired mode:
1. Encrypt — Enter source, destination, and key
2. Decrypt — Enter source, destination, and key
3. Brute-force — Just enter the encrypted file and destination
